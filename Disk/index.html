CKdialog
碎片弹窗 for PHP

CKdialog ,可以在大前端实现弹窗程序碎片化，有利于效率，分工; 

$(".buttonclass").click(function(){
	$.CK({
		rel:$(this).attr('rel'),
	});
});

下载 & 安装

下载压缩包,解压文件,把CKdialog放置到任意的目录下,例如跟下;

第一步,包含文件

<script src="/assets/js/jquery-1.11.1.min.js"></script>
<script src="/assets/js/artDialog4.1.7/artDialog.js?skin=default"></script>
<script src="/assets/js/CK.js"></script><!-- 基于artdialog的组件 -->
<script src="/assets/js/CK_bootstrap.js"></script><!-- 基于bootstrap的组件 -->

下一步:包含CKdialog文件,开始使用. 

$(".buttonclass").click(function(){
	$.CK({
		rel:$(this).attr('rel'),
	});
});
