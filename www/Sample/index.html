CKdialog碎片弹窗 for PHP

下载 & 安装

下载压缩包,解压文件,把CKdialog放置到任意的目录下,例如跟下;

第一步,包含文件

<script type="text/javascript" src="assets/jquery-1.10.2.min.js"></script>
<script type="text/javascript" src="assets/artDialog4.1.7/artDialog.js?skin=default"></script>
<script type="text/javascript" src="assets/CK.js"></script>

下一步:包含CKdialog文件,开始使用. 

$(".buttonclass").click(function(){
	$.CK({
		rel:$(this).attr('rel'),
	});
});

 

代码示例

普通弹窗 

$(".test1").click(function(){
    $.CK({
        rel:$(this).attr('rel')
    });
});
测试

 

普通弹窗 [cancel]按钮

$(".test2").click(function(){
    $.CK({
        cancel:true,
        rel:$(this).attr('rel')
    });
});
测试
